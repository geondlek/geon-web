태그의 종류

  • 피파 바로가기


    • a href="" : 링크다는 태그 (큰따옴표 사이에 링크주소 적으면 됨)
    a href="" target="_blank" : 링크단 태크를 새탭으로 열리게 하는태그
    a href="" target="_blank" title="" : 타이틀안에 적으면 링크에 마우스를 갔다대면 설명해줌


    !doctype html : 모든 페이지 맨위에 붙는 태그
    html : 이것은 html 이라고 나타내는 태그 (꼭 쓰고 시작)
    body : 본문의 시작과 끝에 붙는 태그 (본문 처음과 끝에 꼭 넣어야함 필수!)
    head : 본문을 설명하는 내용에 시작과 끝에 붙는태그 (이것도 처음과 끝에 꼭 넣어야함!)
    br : 줄바꿈태그 (닫는태그 할 필요 없음)
    strong : 글씨진하게 하는 태그
    u : 밑줄태그(영어로 underline)
    h1~h6 : 글씨크기태그 (숫자가높아질수록 글씨는 작아짐)

    p : 단락나누는 태그 (br태그랑 비슷한데 정해진만큼 줄바뀜)





    img : 이미지를 보여주는 태그


    li : 리스트를 나눠주는 태그 (예 : ● 이건)
    ul : li의 부모태그 탭키로 들여쓰기가능
    ol : 앞에 자동으로 숫자 넣어
    title : 페이지 제목 태그
